Compare the results several manifold algorithms on RGB images. SpectralEmbedding, TSNE, Isomap, LocallyLinearEmbedding, MDS, LLE, LTSA, Hessian LLE, and Modified LLE.

### Dependencies:
```
scipy
scikit-image
scikit-learn
numpy
matplotlib
```


## compare_methods.py

### Usage:
```
$ python compare_methods.py
```
